== Краткое описание


Учебный проект для курсов RubyBursa. 
* планируется сделать систему ставок на события
* события не зависят от пользователей
* пользователи делают ставки, система рассчитывает коэффициэнты
* в случае выигрыша пользователь получает баллы к своему рейтингу
* изначально у всех одинаковый рейтинг
